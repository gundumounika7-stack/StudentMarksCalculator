# StudentMarksCalculator
**Author:** Sumanth

Simple Java program that calculates total marks, percentage and grade for a student.

## How to run locally
1. Save `StudentMarksCalculator.java`.
2. Open CMD in the folder.
3. Compile: `javac StudentMarksCalculator.java`
4. Run: `java StudentMarksCalculator`

## Sample output
Name: Sumanth
Total Marks: 433 / 500
Percentage: 86.60%
Grade: B
